# 🌐 Sabrina Arroyo – Portfolio

This is my professional portfolio site, built with **HTML5** and **Bootstrap 5**, and hosted with **GitHub Pages**.  
It highlights my experience in **operations management, vendor coordination, and process development**.  

👉 Live Site: [https://sabrina-arroyo.github.io](https://sabrina-arroyo.github.io)

---

## ✨ Features
- Responsive, mobile-first design using **Bootstrap 5**  
- Sticky navigation bar with smooth scrolling  
- Hero section with profile photo, signature, and contact links  
- Resume download button (PDF)  
- Sections for **Summary, Experience, Skills, Education, and Certifications**
